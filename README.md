# Pinet

An application boilerplate for building RESTful web services with Spring 5 and Java.

[![Build Status](https://travis-ci.org/Chatatata/Pinet.svg?branch=master)](https://travis-ci.org/Chatatata/Pinet)

## Technology stack

- OpenJDK 10
- Spring 5.0.9
- PostgreSQL 10.5
